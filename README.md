# 🤖 AI Resume Analyzer & Interview Coach

## 📌 Overview

AI Resume Analyzer & Interview Coach is a Streamlit-based web application that analyzes resumes, extracts important information, calculates a resume score, and generates interview questions.

---

## ✨ Features

- Upload Resume (PDF)
- Extract Resume Text
- Extract Name, Email & Phone
- Detect Technical Skills
- Resume Score Calculation
- Resume Improvement Suggestions
- AI Interview Question Generator
- Interview Answer Evaluation
- SQLite Database
- Admin Dashboard

---

## 🛠 Technologies Used

- Python
- Streamlit
- SQLite
- PyPDF2
- Pandas

---

## 🚀 How to Run

```bash
pip install -r requirements.txt

streamlit run app.py
```
